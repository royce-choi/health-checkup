# 건강검진 추적 관리

회사 지원 건강검진 병원 조사 및 개인 건강검진 결과를 추적 관리하는 저장소입니다.

## 프로젝트 배경

- 회사 지원 건강검진 패키지 활용 (30만원 기준)
- 신도림역 기준 + 판교 출근길 접근성 좋은 병원 조사
- 매년 검진 결과를 추적하여 건강 변화를 모니터링

## 작업 내역

### 2026-03-09: 초기 조사 완료

**요청**: 신도림역 근처에서 회사 지원 건강검진 병원을 조사하고, 개인 검진결과를 분석하여 2026년 검사 추천

**진행 과정**:
1. 회사 지원 76개 병원 엑셀 데이터에서 신도림역 30분 이내 + 무환승 병원 필터링
2. 후보 5개 병원의 검진항목(30만원 패키지) 상세 비교
3. 웹 검색을 통한 병원별 후기/평판/시설/장비/전문분야 조사
4. 2024년(메디피움 판교), 2025년(미래의료재단) 검진결과 PDF 분석
5. 연도별 건강 수치 변화 추적 및 2026년 맞춤 검사 추천

**분석 결과 요약**:
- 신도림 무환승 5곳 + 환승 19곳 + 판교/분당 9곳 = 총 33개 병원 조사
- TOP 5: 소중한메디케어, 필립메디컬 서울, 아폴로헬스케어, 서울센트럴, 녹십자아이메드
- 아폴로헬스케어: MRI기본 + 피부시술(슈링크/보톡스) 선택 가능한 독특한 옵션
- 건강: ALT·요산·RF 정상화 / 추적필요: WBC 저하, 빌리루빈 상승, 전립선 석회화
- 2026 핵심: ①WBC 추적 ②전립선 추적 ③경추/요추 MRI ④폐CT
