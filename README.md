# 건강검진 추적 관리

회사 지원 건강검진 병원 조사 및 개인 건강검진 결과를 추적 관리하는 저장소입니다.

## 프로젝트 배경

- 회사 지원 건강검진 패키지 활용 (30만원 기준)
- 신도림역 기준 + 판교 출근길 접근성 좋은 병원 조사
- 매년 검진 결과를 추적하여 건강 변화를 모니터링

## 디렉토리 구조

```
health-checkup/
├── README.md                  # 프로젝트 소개
├── reports/
│   ├── 2026-final-recommendation.md  # 최종 추천 및 병원별 최적 조합
│   ├── 2026-health-analysis.md       # 개인 검진결과 분석 및 2026 추천
│   ├── 2026-nearby-hospitals.md      # 신도림 근거리 병원 (무환승 30분 이내)
│   ├── 2026-transfer-hospitals.md    # 서울 환승 병원 (환승1회 40분 이내)
│   └── 2026-pangyo-hospitals.md      # 판교/분당 병원 + 아폴로 피부시술
├── results/
│   ├── 2024-검진결과.pdf             # 2024년 검진결과 원본
│   ├── 2025-검진결과.pdf             # 2025년 검진결과 원본
│   └── *.xlsx                     # 병원 리스트 원본 데이터
└── CHANGELOG.md               # 작업 이력
```

## 리포트 읽는 순서

1. **[건강 분석](reports/2026-health-analysis.md)** - 내 건강 상태와 올해 받아야 할 검사 확인
2. **[최종 추천](reports/2026-final-recommendation.md)** - TOP 5 병원 + 병원별 최적 조합
3. 병원 상세가 궁금하면: [근거리](reports/2026-nearby-hospitals.md) | [환승](reports/2026-transfer-hospitals.md) | [판교](reports/2026-pangyo-hospitals.md)
