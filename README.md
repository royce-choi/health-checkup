# 건강검진 추적 관리

회사 지원 건강검진 병원 조사 및 개인 건강검진 결과를 추적 관리하는 저장소입니다.

## 프로젝트 배경

- 회사 지원 건강검진 패키지 활용 (30만원 기준)
- 신도림역 기준 접근성 좋은 병원 조사
- 매년 검진 결과를 추적하여 건강 변화를 모니터링

## 디렉토리 구조

```
health-checkup/
├── README.md                  # 프로젝트 소개
├── reports/
│   ├── 2026-hospital-research.md   # 2026년 병원 조사 및 비교 분석
│   └── 2026-health-analysis.md     # 개인 검진결과 분석 및 2026 추천
├── results/
│   ├── 2024-검진결과.pdf             # 2024년 검진결과 원본
│   ├── 2025-검진결과.pdf             # 2025년 검진결과 원본

└── CHANGELOG.md               # 작업 이력
```

## 작업 내역

### 2026-03-09: 초기 조사 완료

**요청**: 신도림역 근처에서 회사 지원 건강검진 병원을 조사하고, 개인 검진결과를 분석하여 2026년 검사 추천

**진행 과정**:
1. 회사 지원 76개 병원 엑셀 데이터에서 신도림역 30분 이내 + 무환승 병원 필터링
2. 후보 5개 병원의 검진항목(30만원 패키지) 상세 비교
3. 웹 검색을 통한 병원별 후기/평판/시설/장비/전문분야 조사
4. 2024년(메디피움 판교), 2025년(미래의료재단) 검진결과 PDF 분석
5. 연도별 건강 수치 변화 추적 및 2026년 맞춤 검사 추천

**분석 결과 요약**:
- 후보 병원: 필립메디컬(7분), 영등포병원(12분), 소중한메디케어(12분), 하나로의료재단(24분), 녹십자아이메드(27분)
- 1순위 추천: 필립메디컬 서울 (가까움+품질), 2순위: 소중한메디케어 (MRI기본+가성비)
- 건강 호전: ALT·요산·RF 정상화, 위내시경 호전, 체중감량
- 추적 필요: WBC 지속 저하, 간접빌리루빈 상승, 전립선 석회화
- 2026 핵심: ①WBC 추적 ②전립선 추적 ③경추/요추 MRI 첫 시도 ④폐CT 첫 시도
