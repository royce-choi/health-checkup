## 올해 검진 우선순위

> - **필수**: 위내시경(수면), 혈액검사(WBC추적), 간기능(ALT/ALP/빌리루빈), 전립선초음파, 요산
> - **추적 권장**: 상복부초음파, 청력검사, RF(류마티스)
> - **올해 새로**: 경추/요추 MRI, 폐CT
> - **안 받아도 됨**: 대장내시경(2025정상), 뇌MRI(2025정상), 심장초음파(2025정상)

## TOP 5 추천 병원 + 최적 조합

### 1위: 소중한메디케어 ★★★★★ (가산, 12분, 무환승)

> MRI + 폐CT + 위내시경 모두 해결. 가성비 최강.

| 구분 | 선택 | 이유 |
| --- | --- | --- |
| **근골격 기본** | 경추CT, 요추CT | 기본 제공 |
| **일반장비 기본** | 전립선초음파, 상복부초음파, 갑상선초음파 | 모두 기본! |
| **E그룹 택1** | **위내시경(수면)** | ← 위내시경이 E그룹에 있음! |
| **A그룹 택1** | **경추MRI-T2** | 올해 첫 시도, 디스크 평가 |
| **B그룹 택1** | **흉부(폐)CT** | 올해 첫 시도, 조기 폐암 발견 |
| **C그룹 택1** | **NK세포 활성도** or 활성산소 검사 | 대장내시경 불필요하므로 |
| **D그룹 택1** | **골밀도검사** or Anti-CCP | 류마티스 추적 |
| **3월 프로모션** | CT 1부위 무료 | B그룹 폐CT와 별도로 추가 CT 가능! |

**총 검사**: 위내시경 + 경추MRI + 폐CT + 전립선초음파 + 상복부초음파 + 갑상선초음파 + 경추CT + 요추CT + NK세포 + 골밀도 + 혈액검사 + **프로모션CT**
**아쉬운 점**: 요추MRI 못 받음(A그룹 택1). 대기시간 매우 김.
**팁**: 반드시 3월 내 평일 이른 아침 예약!

### 2위: 아폴로헬스케어 ★★★★ (판교2밸리, 출근길)

> MRI기본 + 피부시술 + 폐CT. 올해 대장내시경 불필요하니 슈링크 찬스!

| 구분 | 선택 | 이유 |
| --- | --- | --- |
| **근골격 기본** | **경추MRI-T2, 요추MRI-T2** + 경추CT, 요추CT | MRI+CT 모두 기본! |
| **일반장비 기본** | 전립선초음파, 상복부초음파, 갑상선초음파, 골밀도 | 모두 기본! |
| **A그룹 택1** | **위내시경(수면)** | 필수 |
| **B그룹 택1** | **슈링크(200샷)** | 대장내시경 불필요→피부시술! |
| **C그룹 택1** | **경동맥초음파+동맥경화** or **NK세포 활성도** | 추적 검사 |
| **D그룹 택1** | **흉부(폐)CT** | 올해 첫 시도 |
| **E그룹 택1** | **Cyfra21-1(폐암)** or 호모시스테인 | 폐CT와 세트 |

**총 검사**: 위내시경 + **경추MRI + 요추MRI** + 폐CT + 전립선초음파 + 상복부초음파 + 갑상선초음파 + 경추CT + 요추CT + 골밀도 + 경동맥초음파 + 혈액검사 + **슈링크200샷**
**최강 구성**: 경추/요추 MRI 따로 받으면 30~60만원인데 기본 포함 + 폐CT + 슈링크까지.
**아쉬운 점**: 판교2밸리 위치(1밸리에서 15~20분). 후기 데이터 부족.
**대안**: B그룹에서 슈링크 대신 뇌혈관MRA or 간MRI 선택 가능.

### 3위: 필립메디컬 서울 ★★★★ (당산, 7분, 무환승)

> 가장 가까움 + 대기 짧음 + 영상판독 최상. 효율 중시.

| 구분 | 선택 | 이유 |
| --- | --- | --- |
| **근골격 기본** | 경추CT, 요추CT | 기본 제공 |
| **일반장비 기본** | 전립선초음파, 상복부초음파, 갑상선초음파, 골밀도 | 모두 기본! |
| **A그룹 택1** | **위내시경(수면)** | 필수 |
| **B그룹 택2** | **흉부(폐)CT** + **경동맥초음파(~8월 무료)** | 폐CT 첫 시도 |
| **프로모션** | 경동맥초음파 무료(남성, ~8월) | B그룹 선택과 별개로 무료! |

**총 검사**: 위내시경 + 폐CT + 전립선초음파 + 상복부초음파 + 갑상선초음파 + 경추CT + 요추CT + 골밀도 + 경동맥초음파(무료) + 혈액검사
**아쉬운 점**: MRI 없음(CT만). 선택 3개로 적음.
**장점**: 3.0T MRI + 64채널 MDCT, 영상의학 전문의 8명 판독. 7분 거리.

### 4위: 서울센트럴 메디케어 ★★★★ (마곡, 30분, 환승1회)

> 경추MRI 기본 + 요추MRI 선택 + 폐CT + 5그룹 선택. 밸런스형.

| 구분 | 선택 | 이유 |
| --- | --- | --- |
| **근골격 기본** | **경추MRI** + 경추CT, 요추CT | 경추MRI 기본! |
| **일반장비 기본** | 전립선초음파, 상복부초음파, 갑상선초음파, 골밀도 | 모두 기본! |
| **A그룹 택1** | **위내시경(수면)** | 필수 |
| **B그룹 택1** | **요추MRI** | 경추는 기본, 요추도 MRI로! |
| **C그룹 택1** | **흉부(폐)CT** | 올해 첫 시도 |
| **D그룹 택1** | 대장내시경 전용 → **패스** | 올해 불필요 |
| **E그룹 택1** | **경동맥초음파** or 동맥경화검사 | 추적 |

**총 검사**: 위내시경 + **경추MRI + 요추MRI** + 폐CT + 전립선초음파 + 상복부초음파 + 갑상선초음파 + 경추CT + 요추CT + 골밀도 + 경동맥초음파 + 혈액검사
**아쉬운 점**: D그룹 대장내시경 전용이라 1개 낭비. 이동 30분.
**프로모션**: 3월 동맥경화+수액+독감 무료.

### 5위: 녹십자아이메드 강남 ★★★★ (강남역, 34분, 무환승)

> A택3 = 선택폭 압도적. MRI 기본.

| 구분 | 선택 | 이유 |
| --- | --- | --- |
| **근골격 기본** | 경추CT, 요추CT + **경추MRI, 요추MRI** | MRI도 기본! |
| **일반장비 기본** | 유방X선, 심전도, 흉부X선 | 초음파는 A그룹에서 |
| **A그룹 택3** | **위내시경(수면)** + **전립선초음파** + **상복부초음파** | 필수 3개 해결 |
| **B그룹 택1** | **흉부(폐)CT** | 올해 첫 시도 |
| **C그룹 택1** | **NK세포 활성도** or 알러지검사108종 | 대장내시경 불필요 |

**총 검사**: 위내시경 + **경추MRI + 요추MRI** + 폐CT + 전립선초음파 + 상복부초음파 + 경추CT + 요추CT + NK세포 + 혈액검사
**아쉬운 점**: 갑상선초음파 기본에 없음. 이동 34분.
**장점**: 2020년 신축 2,200평 최신 시설.

### HS한신메디피아 ★★★☆ (잠원, 35분, 환승1회)

> 경추/요추 MRI-T2 기본 + B택2 + C택2 = 선택 5개. 강남 최대규모.

**선택**: A:위내시경 / B:폐CT+경동맥초음파 / C:골밀도+동맥경화
**총 검사**: 위내시경 + 경추MRI + 요추MRI + 폐CT + 전립선초음파 + 상복부초음파 + 갑상선초음파 + 경추CT + 요추CT + 경동맥초음파 + 골밀도 + 동맥경화 + 혈액검사
**장점**: MRI기본 + 선택5개. AI맞춤검진.  |  **아쉬운 점**: 이동 35분.

### 지앤이알파돔 ★★★☆ (판교 도보권)

> 판교 출근 전 검진. 영상의학과 전문의 상주로 당일 소견 상담.

**선택**: A:위내시경 / B:전립선초음파 / C:폐CT+경추CT / D:뇌MRI
**총 검사**: 위내시경 + 폐CT + 전립선초음파 + 상복부초음파 + 갑상선초음파 + 경동맥초음파(기본!) + 경추CT + 요추CT + 뇌MRI + 혈액검사
**장점**: 도보권, 영상의학과 당일 상담.  |  **아쉬운 점**: MRI 없음(+15만원 프로모).

> **최종 권고**: 올해는 경추/요추 MRI + 폐CT를 처음 받는 해. 대장내시경은 안 받아도 되니 선택 여유가 있음.
>
> - **가성비 최강** = 소중한메디케어 (3월 프로모션 놓치지 말 것!)
> - **MRI+피부시술** = 아폴로헬스케어 (슈링크 받기 최적의 해)
> - **편의성 최강** = 필립메디컬 서울 (7분+대기짧음, MRI없는게 유일한 약점)
